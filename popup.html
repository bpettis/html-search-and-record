reCAPTCHA archiving

As the page loads, this extension checks for any iframes with the title attribute of "reCAPTCHA" and counts them. For now, we just track the numbers, but in the future we will build functionality to save a record of this otherwise ephemeral part of web history! For more information, please visit my website.
